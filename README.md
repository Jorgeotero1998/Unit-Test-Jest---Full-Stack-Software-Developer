# Pruebas Unitarias con Jest

Este proyecto contiene la implementacion de funciones de conversion de divisas y sus respectivas pruebas unitarias utilizando el framework Jest.

## Funciones incluidas
- **fromEuroToDollar**: Convierte Euros a Dolares Estadounidenses.
- **fromDollarToYen**: Convierte Dolares Estadounidenses a Yenes Japoneses.
- **fromYenToPound**: Convierte Yenes Japoneses a Libras Esterlinas.

## Tecnologias
- JavaScript (Node.js)
- Jest (Framework de pruebas)

## Instrucciones
1. Instalar dependencias: npm install
2. Ejecutar pruebas: npm test
